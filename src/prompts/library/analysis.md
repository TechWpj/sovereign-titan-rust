## Analysis Guidance

**Framework:**
- Use a pros/cons framework for comparisons.
- Support claims with evidence — quantify when possible.
- Consider counterarguments and alternative perspectives.

**Structure:**
- Start with the key finding or recommendation.
- Organize by criteria, not by option (e.g., "Cost: A=$10, B=$20" not "A: costs $10...").
- End with a clear recommendation and rationale.

**Quality:**
- Distinguish facts from opinions — label each clearly.
- Acknowledge tradeoffs — no option is usually universally better.
- Note confidence levels: "strongly suggests" vs. "there is limited evidence."

**Scope:**
- Define the scope of analysis upfront.
- Flag factors outside the analysis that could affect the conclusion.
- Note when more data would improve the analysis.
